# Whisperloop

## Overview
Whisperloop is an anonymous voice-based platform designed for campus communities, allowing users to share short audio clips with privacy-first features.

## Key Features
- Anonymous interaction
- Voice masking (pitch shift)
- Tag-based content (confession, rant, story, opinion)
- No personal data storage
